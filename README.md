# Exercicio — Como pôr o projeto a correr 

Este documento descreve como pôr o projeto a correr localmente.

## Passo-a-passo (IDE)

1) Executar a aplicação (run)

- Localize a classe `ExercicioApplication` em `src/main/java/com/example/exercicio/ExercicioApplication.java` e execute-a (Run ▶).

2) Abrir o Swagger UI

- No navegador, aceda a: `http://localhost:8080/swagger-ui/index.html`

## Testes Unitarios
- Para correr os testes unitários, no IDE clique com o direito sobre a pasta `src/test/java` e escolha "Run".
